# Instalasi final GitHub Profile

1. Buka repository `GhoziTech/GhoziTech`.
2. Hapus `README.md` lama dan folder `assets` lama.
3. Upload `README.md` dan seluruh folder `assets` dari paket ini.
4. Commit perubahan.
5. Buka profil GitHub lalu tekan `Ctrl + F5`.

README menggunakan `dark.webp` dan `light.webp` agar gambar serta animasi tampil stabil di GitHub. File `dark.svg` dan `light.svg` tetap disertakan sebagai source code editable.
